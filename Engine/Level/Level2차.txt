Level class
- engine에서 이제 actor를 그리게 되면서 actor를 다루기 위한 무대인 level을 구현하게 되었다 
- 그동안 미구현으로 남아있던 함수들의 내용을 추가하였다

- 스페이스바 입력으로 화면에 추가 그릴 대상이 추가 되도록 하였다
- 이에 따라 이것을 관리하기 위한 함수가 탄생하였다

Draw 함수
- Actor Class 에서 고민했듯, 같은 위치에 다른 액터가 있는지 확인하는 것이 중요하다
- 이중 반복문을 수행하여 동일 객체는 검사하지 않도록 한다
- 그렇게 해서 객체들이 서로 다른 존재라면, Coord 값을 비교하여 같은지 다른지 비교한다
- 좌표가 다르다면 문제 없이 그리고, 같다면 Actor의 SortingOrder 값에 따라 우선순위가 높은 Actor를 그린다

AddNewActor 함수
- Actor Class 에 있떤 오너쉽이 들어간다... 솔직히 잘 모르겠따!
- vector의 emplace_back에 대해 알아보자!

ProcessAddAndDestroyActors 함수
- Actor 배열을 처리하는 함수로 제거 및 추가를 하는 역할을 한다
- 모든 액터들을 돌아다니며 DestroyRequested()의 값으로 제거 유무에 따라 제거를 진행한다
- 해당 프로젝트에서는 동적 할당을 하였기에 delete를 한 후, vector의 erase를 통해 해당 부분을 제거한다!
- 추가할 배열이 있다면 추가를 하는데, size 값을 통해 가치를 판단한다
- 만약 추가할 배열이 있다면 for문(뭐라고 하는지 모르겠지만)을 통해 순차적으로 emplace_back을 적용한다
